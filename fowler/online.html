Fowler

The need

Fowler is an encryption and decryption tool using Web cryptography technologies. Its usage of experimental technologies also implies that this tool will only work on the latest browser versions and might stop working in the future whenever one of the experimental technologies gets deprecated.

It is not recommended using this tool for sensitive matters as this interface exchanges your texts with an application I host online, i.e. you would need to trust me and my service provider. Not only that but it is impossible to guarantee messages cannot be decrypted outside of the application.

Hot it works

To encrypt a message, input your encryption key and your text in the form below, then click on "Encrypt". The key and the message will be sent to an environment that exists only during the time it takes to encrypt and return your message. The encrypted message will be returned on the readonly box below. Clicking that box will automatically select the text for you. Do not forget to send both the key and the encrypted message to your recipient. Separately, if possible.

To decrypt a message, input the encrypted text and the decryption key that have been sent to you by the sender, then click on "Decrypt". The key and the encrypted message will be sent to an environment that exists only during the time it takes to decrypt and return your message. The decrypted message will be returned on the readonly box below. Clicking that box will automatically select the text for you, even though this should not matter.
